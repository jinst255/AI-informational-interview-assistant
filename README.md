# AI Informational Interview Assistant

An AI-powered assistant that helps run informational interviews by capturing notes, structuring questions, and keeping interviews on track.

**Problem it solves:** Manual note-taking and interview prep and aftermath are messy and inconsistent. I wanted to streamline the whole process.

**What this app will do:**
- Guide interviews with a focused question flow
- Capture and organize notes in real time
- Produce a clean summary for follow-up actions
